## 使用
在config/config.py里面配置cookies 
```
{
    'OUTFOX_SEARCH_USER_ID': '',
    'OUTFOX_SEARCH_USER_ID_NCOO': '',
    '_uetsid': '',
    '_uetvid': '',
    'DICT_DOCTRANS_SESSION_ID': '',
}
```
安装所需的第三方包   
```angular2html
pip install -r requirements.txt
```
运行main.py
